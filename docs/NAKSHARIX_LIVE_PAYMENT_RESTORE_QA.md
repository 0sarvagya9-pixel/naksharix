## Vercel Webhook Dashboard Setup Requirement
> [!IMPORTANT]
> **WEBHOOK_CONFIG_REQUIRED_IN_RAZORPAY_DASHBOARD**<br />
> To complete live webhook event delivery:
> 1. Set up a Webhook URL in your Razorpay Dashboard pointing to: `https://<your-domain>/api/payments/razorpay/webhook`
> 2. Configure active events: `payment.captured` and `payment.failed`.
> 3. Save the secret generated by Razorpay and configure it as `RAZORPAY_WEBHOOK_SECRET` in your Vercel project settings.
